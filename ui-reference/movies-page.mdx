# MoviesPage

The `MoviesPage` is a placeholder page that will eventually display movie-related content. Currently, it shows a simple centered text label.

## Overview

`MoviesPage` is a stateless widget that serves as one of the three main navigation destinations in the app. It's accessible from the bottom navigation bar on the `HomePage`.

## Implementation

```dart
import 'package:flutter/material.dart';

class MoviesPage extends StatelessWidget {
  const MoviesPage({super.key});

  @override
  Widget build(BuildContext context) {
    return const Scaffold(
      body: Center(
        child: Text('Movies'),
      ),
    );
  }
}
```

## Current Features

### Simple Layout

- **Scaffold**: Provides basic page structure
- **Centered Text**: Displays "Movies" as a placeholder

### Stateless Design

As a placeholder, this page doesn't require state management and is implemented as a `StatelessWidget`.

## Navigation Flow

```
HomePage → MoviesPage
          (via bottom navigation - Index 2)
```

Users access this page by tapping the "Movies" tab (with `Icons.movie` icon) in the bottom navigation bar.
