# ComingSoonPage

The `ComingSoonPage` is a placeholder page designed to display upcoming game releases and future content. Currently, it shows a simple centered text label.

## Overview

`ComingSoonPage` is a stateless widget that serves as one of the three main navigation destinations in the app. It's accessible from the bottom navigation bar on the `HomePage`.

## Implementation

```dart
import 'package:flutter/material.dart';

class ComingSoonPage extends StatelessWidget {
  const ComingSoonPage({super.key});

  @override
  Widget build(BuildContext context) {
    return const Scaffold(
      body: Center(
        child: Text('Coming Soon'),
      ),
    );
  }
}
```

## Current Features

### Simple Layout

- **Scaffold**: Provides basic page structure
- **Centered Text**: Displays "Coming Soon" as a placeholder

### Stateless Design

As a placeholder, this page doesn't require state management and is implemented as a `StatelessWidget`.

## Navigation Flow

```
HomePage → ComingSoonPage
          (via bottom navigation - Index 1)
```

Users access this page by tapping the "Coming Soon" tab (with `Icons.new_releases` icon) in the bottom navigation bar.
