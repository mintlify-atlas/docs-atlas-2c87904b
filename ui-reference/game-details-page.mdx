# GameDetailsPage

The `GameDetailsPage` provides a comprehensive view of a specific game, including its header image, description, action buttons (Rent/Buy), and a horizontal gallery of screenshots.

## Overview

`GameDetailsPage` is a stateful widget that receives a `Game` object and displays its complete details in a scrollable layout. The page includes interactive buttons for renting or purchasing the game.

## Implementation

```dart
import 'package:flutter/material.dart';
import 'package:flutter_games/component/item_description.dart';
import 'package:flutter_games/component/item_header_game.dart';
import 'package:flutter_games/controller/scroll_horizontal_screenshots.dart';
import 'package:flutter_games/model/game.dart';

class GameDetailsPage extends StatefulWidget {
  GameDetailsPage(this.game, {Key? key}) : super(key: key);

  final Game game;

  @override
  _GameDetailsPageState createState() => _GameDetailsPageState();
}

class _GameDetailsPageState extends State<GameDetailsPage> {
  @override
  Widget build(BuildContext context) {
    return Material(
      borderRadius: BorderRadius.circular(8),
      clipBehavior: Clip.hardEdge,
      child: SingleChildScrollView(
        child: Column(
          children: [
            GameDetailHeader(widget.game),
            // Rent Button
            Padding(
              padding: const EdgeInsets.fromLTRB(16, 32, 16, 0),
              child: SizedBox(
                width: double.infinity,
                child: ElevatedButton(
                  onPressed: () => {},
                  child: Row(
                    mainAxisAlignment: MainAxisAlignment.center,
                    children: <Widget>[
                      Padding(
                        padding: const EdgeInsets.only(right: 8),
                        child: Icon(Icons.adjust, color: Colors.white),
                      ),
                      Text(
                        "Rent",
                        style: Theme.of(context)
                            .textTheme
                            .titleMedium!
                            .apply(color: Colors.white),
                      ),
                    ],
                  ),
                  style: ButtonStyle(
                    padding: MaterialStateProperty.all(
                        const EdgeInsets.all(12)),
                    shape: MaterialStateProperty.all(
                        RoundedRectangleBorder(
                      borderRadius: BorderRadius.circular(4),
                    )),
                    backgroundColor: MaterialStateProperty.all(
                        Colors.green),
                    overlayColor: MaterialStateProperty.resolveWith<Color?>(
                      (Set<MaterialState> states) {
                        if (states.contains(MaterialState.pressed))
                          return Colors.green.shade400;
                        return null;
                      },
                    ),
                    elevation: MaterialStateProperty.all(8),
                    shadowColor: MaterialStateProperty.all(
                        Colors.green.shade400),
                  ),
                ),
              ),
            ),
            // Buy Button
            Padding(
              padding: const EdgeInsets.fromLTRB(16, 16, 16, 0),
              child: SizedBox(
                width: double.infinity,
                child: OutlinedButton(
                  onPressed: () => {},
                  child: Row(
                    mainAxisAlignment: MainAxisAlignment.center,
                    children: <Widget>[
                      Padding(
                        padding: const EdgeInsets.only(right: 8),
                        child: Icon(Icons.archive, color: Colors.green),
                      ),
                      Text(
                        "Buy",
                        style: Theme.of(context)
                            .textTheme
                            .titleMedium!
                            .apply(color: Colors.green),
                      ),
                    ],
                  ),
                  style: ButtonStyle(
                    padding: MaterialStateProperty.all(
                        const EdgeInsets.all(12)),
                    shape: MaterialStateProperty.all(
                        RoundedRectangleBorder(
                      borderRadius: BorderRadius.circular(4),
                    )),
                    side: MaterialStateProperty.all(
                        BorderSide(color: Colors.green, width: 4)),
                    backgroundColor: MaterialStateProperty.all(
                        Colors.white),
                    overlayColor: MaterialStateProperty.resolveWith<Color?>(
                      (Set<MaterialState> states) {
                        if (states.contains(MaterialState.pressed))
                          return Colors.green.shade200;
                        if (states.contains(MaterialState.hovered) ||
                            states.contains(MaterialState.focused))
                          return Colors.white70;
                        return null;
                      },
                    ),
                    elevation: MaterialStateProperty.all(0),
                    foregroundColor: MaterialStateProperty.all(
                        Colors.green.shade400),
                  ),
                ),
              ),
            ),
            Padding(
                padding: const EdgeInsets.fromLTRB(16, 24, 16, 0),
                child: DescriptionText(widget.game.description)),
            HorizontalScreenshotController(widget.game.screenshots),
          ],
        ),
      ),
    );
  }
}
```

## Key Features

### Page Structure

The page is wrapped in a `Material` widget with rounded corners and uses `SingleChildScrollView` to allow vertical scrolling through all content.

### Components

1. **GameDetailHeader**: Displays the game's header image and primary information
2. **Rent Button**: Green elevated button with shadow for renting the game
3. **Buy Button**: Green outlined button for purchasing the game
4. **DescriptionText**: Shows the game's detailed description
5. **HorizontalScreenshotController**: Horizontal scrolling gallery of game screenshots

### Action Buttons

#### Rent Button
- **Style**: Elevated button with green background
- **Icon**: `Icons.adjust` in white
- **Features**: 
  - 8px elevation with green shadow
  - Rounded corners (4px radius)
  - Press effect with lighter green shade
  - Full width layout

#### Buy Button
- **Style**: Outlined button with white background
- **Icon**: `Icons.archive` in green
- **Features**:
  - 4px green border
  - Rounded corners (4px radius)
  - No elevation (flat appearance)
  - Press and hover effects
  - Full width layout

### Layout Details

- **Border Radius**: 8px on the Material wrapper
- **Padding**: Consistent 16px horizontal padding throughout
- **Spacing**: 32px top padding for Rent button, 16px between buttons, 24px before description
