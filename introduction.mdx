# Introduction to Flutter Games

Flutter Games is a demo application showcasing Flutter's capabilities for building beautiful, responsive game marketplace apps. This app demonstrates best practices for creating elegant UIs with smooth animations, hero transitions, and a delightful user experience.

## What You'll Build

Flutter Games includes three main sections accessible via bottom navigation:

- **Games** - Browse and purchase games with detailed views including screenshots, ratings, and descriptions
- **Coming Soon** - Preview upcoming game releases
- **Movies** - Explore movie rentals and purchases

## Key Features

### Elegant Image Loading
Flutter intelligently prioritizes visible images in scrollable lists, ensuring smooth performance and optimal loading behavior.

### Custom Game Cards
Beautifully designed game cards display:
- High-quality cover images
- Star ratings
- Pricing information
- Game titles and descriptions

### Minimal Dependencies
The app demonstrates how much can be accomplished with Flutter's built-in capabilities:

```yaml lib/pubspec.yaml
dependencies:
  flutter:
    sdk: flutter
  cupertino_icons: ^1.0.8
  responsive_framework: ^1.5.1
  google_fonts: ^6.3.1
```

## Architecture

The app follows a clean architecture with organized folders:

- `lib/pages/` - Main page components (home, games, movies, coming soon)
- `lib/widgets/` - Reusable UI components like game cards
- `lib/models/` - Data models for games and content
- `lib/themes.dart` - Theme configuration for light and dark modes
- `lib/utils/` - Utility functions and dummy data

## Technology Stack

- **Framework**: Flutter 3.10.0+
- **Language**: Dart 3.0.0+
- **State Management**: StatefulWidget
- **UI Components**: Material Design
- **Typography**: Google Fonts (Lato)
- **Responsive Design**: Responsive Framework
